# weather-card-skill

Generate weather cards using Gemini via CDP.

## Installation

```bash
npx skills add Lumiaqian/weather-card-skill -g -y
```

Or install gemini-web-cli directly:

```bash
npm install -g @lumiaqian/gemini-web-cli
```

## Usage

```
weather card → get data → select style → generate image
```

See [SKILL.md](./SKILL.md) for full documentation.
