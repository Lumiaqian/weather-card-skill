# Weather Card Style Guide

## Three Styles

### App Style (信息清晰)

**Characteristics**:
- Clear hierarchy, minimal decoration
- Data-focused, numbers prominent
- Professional color scheme
- No unnecessary elements

**Color Palette**:
- Primary: #4A90D9 (sky blue)
- Secondary: #6B7280 (gray)
- Background: #F8FAFC (off-white)
- Text: #1F2937 (dark gray)
- Accent: #10B981 (green for good weather)

**Typography**:
- Sans-serif (Inter, Roboto)
- Bold temperature display
- Regular weight for details

### Social Style (社媒卡片)

**Characteristics**:
- Eye-catching, title-focused
- Soft, warm colors
- Romantic/moody atmosphere
- Lifestyle feel

**Color Palette**:
- Primary: #F59E0B (warm amber)
- Secondary: #EC4899 (pink)
- Background: linear-gradient sunset tones
- Text: #FFFFFF (white with shadow)

**Typography**:
- Display font for temperature
- Handwritten style for tagline
- High contrast for readability

### Xiaohongshu Style (小红书风)

**Characteristics**:
- Soft, creamy pastels
- Cute sticker-like labels
- Lifestyle blogger tone
- Playful but refined

**Color Palette**:
- Primary: #FBBF24 (soft yellow)
- Secondary: #F472B6 (pink)
- Background: #FEF3C7 (cream)
- Accent: #A78BFA (lavender)
- Text: #92400E (warm brown)

**Typography**:
- Rounded, friendly fonts
- Emoji integration
- Badge-style labels
